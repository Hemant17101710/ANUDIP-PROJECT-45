A simple Banking System Management project developed in Python using Object-Oriented Programming (OOP) concepts and JSON file storage.

## Features

- Create New Account
- Secure Login using PIN
- Deposit Money
- Withdraw Money
- Check Account Balance
- Transfer Money
- Transaction History
- Update Account Details
- Close Account
- Data stored permanently using JSON

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON File Handling

1. Clone the repository


2. Open project directory

```bash
cd Banking-System
```

3. Run

```bash
python main.py
```
